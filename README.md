# bpmn-txt

Text DSL for BPMN 2.0 — human-readable process definitions.

Write BPMN processes in plain text, get valid BPMN 2.0 XML out.

## Install

```bash
npm install @cwygoda/bpmn-txt
```

## VS Code Extension

Full language support: syntax highlighting, diagnostics, autocomplete, go-to-definition, and live diagram preview.

1. Download the `.vsix` from the [latest release](https://github.com/cwygoda/bpmn-txt/releases/latest)
2. In VS Code: Extensions → `...` menu → **Install from VSIX...**
3. Select the downloaded `.vsix` file

See [vscode-bpmn-txt/README.md](vscode-bpmn-txt/README.md) for full feature docs.
